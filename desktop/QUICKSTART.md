# Быстрый старт - AI Mentor Desktop

## Требования

Перед началом убедитесь, что установлено:

1. **Node.js 18+** - https://nodejs.org/
2. **Python 3.11+** - https://www.python.org/downloads/
3. **Ollama** - https://ollama.com/download

## Быстрая сборка (3 команды)

Откройте терминал в папке `ai-mentor`:

```bash
# 1. Переходим в desktop
cd desktop

# 2. Устанавливаем зависимости
npm install

# 3. Запускаем полную сборку
npm run build:win
```

**Или просто запустите:** `build.bat` (двойной клик)

## Результат

После сборки в папке `desktop/dist/` появится:

```
AI Mentor Setup 1.0.0.exe
```

Это готовый установщик Windows.

## Установка приложения

1. Запустите `AI Mentor Setup 1.0.0.exe`
2. Выберите директорию установки
3. Запустите AI Mentor с рабочего стола

## Первый запуск

При первом запуске приложение:
1. Проверит наличие Ollama
2. Загрузит AI модель qwen2.5-coder:7b (~4 ГБ)
3. Запустит FastAPI сервер
4. Откроет главное окно

## Как пользоваться

- **Открыть приложение** - ярлык на рабочем столе
- **Свернуть** - закройте окно (приложение останется в трее)
- **Открыть из трея** - клик по иконке в трее
- **Выход** - правый клик на трее, затем "Выход"

## Решение проблем

### "Python не найден"

```bash
python --version
```

Если ошибка - добавьте Python в PATH Windows.

### "Ollama не установлен"

Скачайте с https://ollama.com/download

### Фронтенд не собирается

```bash
cd ../frontend
npm install
npm run build
cd ../desktop
```
