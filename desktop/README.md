# AI Mentor Desktop

**Полноценное десктопное приложение AI Mentor для Windows**

Локальный AI-наставник по программированию, который работает напрямую на вашем компьютере. Полная конфиденциальность - все данные остаются у вас.

## Возможности

### AI-ментор
- **4 режима**: Обучение, Дебаг, Код-ревью, Практика
- **20+ направлений**: Python, JavaScript, Java, C++, SQL и другие
- Работает **полностью локально** через Ollama
- Не даёт готовых решений - учит думать

### Встроенные инструменты
- **Python REPL** - запуск кода прямо в чате (Pyodide)
- **SQL REPL** - SQL песочница (DuckDB)
- **Библиотека сниппетов** - сохранение кода
- **Проверка решений** - оценка вашего кода

### Система прогресса
- **XP и уровни** - зарабатывайте очки
- **Достижения** - открывайте награды
- **Статистика** - отслеживайте прогресс

### Дизайн
- **3 темы**: тёмная, светлая, AMOLED
- **2 стиля**: Glass Morphism и Brutalist
- Голосовой ввод и вывод
- Экспорт чата в Markdown

### Десктоп функции
- **Установщик Windows** (.exe)
- **Работа в фоне** (системный трей)
- **Автообновления** (каждые 6 часов)
- **Экран первой настройки**
- **Интеграция с Ollama**

## Установка

### Для пользователей

1. **Скачайте установщик**
   - С GitHub Releases: https://github.com/YOUR_USERNAME/ai-mentor/releases
   - Или соберите сами (см. ниже)

2. **Запустите установщик**
   - Дважды кликните `AI Mentor Setup X.X.X.exe`
   - Выберите директорию установки

3. **Первый запуск**
   - Приложение проверит наличие Ollama
   - Загрузит AI модель qwen2.5-coder:7b (~4 ГБ)
   - Готово к работе!

### Для разработчиков

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/YOUR_USERNAME/ai-mentor.git
cd ai-mentor/desktop

# 2. Установите зависимости
npm install

# 3. Сгенерируйте иконки (опционально)
npm run icons

# 4. Запустите в режиме разработки
npm run dev

# 5. Соберите установщик
npm run build:win
```

**Или просто запустите:** `build.bat`

## Как пользоваться

### Основной интерфейс

Приложение открывается как обычное окно. Все функции доступны через UI.

### Системный трей

При закрытии окна приложение **сворачивается в трей** (возле часов):

- **Левый клик** - открыть/скрыть окно
- **Правый клик** - контекстное меню:
  - Открыть AI Mentor
  - Перезапустить
  - Проверить обновления
  - Выход

### Автообновления

- Проверяются автоматически каждые 6 часов
- Загрузка происходит в фоне
- При завершении предлагается перезапуск

## Настройка

### Выбор модели

По умолчанию используется `qwen2.5-coder:7b`. Для изменения:

1. Откройте `ollama-manager.js`
2. Измените `this.modelName`
3. Пересоберите приложение

### Порт бэкенда

По умолчанию: `8000`. Для изменения:

1. Откройте `backend-starter.js`
2. Измените `this.port`
3. Пересоберите

### Частота обновлений

По умолчанию: 6 часов. Для изменения:

1. Откройте `main.js`
2. Найдите `setInterval` (строка ~290)
3. Измените интервал в миллисекундах

## Решение проблем

### "Python не найден"

**Причина:** Python не установлен или не в PATH

**Решение:**
```bash
python --version
```

Если ошибка - переустановите Python с добавлением в PATH.

### "Ollama не установлен"

**Причина:** Ollama не установлен на компьютере

**Решение:**
1. Скачайте с https://ollama.com/download
2. Установите
3. Перезапустите AI Mentor

### Модель не загружается

**Причина:** Проблемы с интернетом или Ollama сервером

**Решение:**
```bash
ollama list
ollama serve
ollama pull qwen2.5-coder:7b
```

### Фронтенд не отображается

**Причина:** Бэкенд не запустился

**Решение:**
1. Проверьте логи в консоли (DevTools)
2. Перезапустите приложение через трей
3. Проверьте, что порт 8000 свободен

## Обновление версии

Для выпуска новой версии:

1. Измените версию в `package.json`:
   ```json
   "version": "1.0.1"
   ```

2. Внесите изменения в код

3. Соберите новой версии:
   ```bash
   npm run build:win
   ```

4. Опубликуйте на GitHub Releases

Автообновления будут работать автоматически.

## Безопасность

- **contextIsolation: true** - изоляция контекста
- **nodeIntegration: false** - нет доступа к Node.js
- **Preload скрипт** - безопасный IPC
